The website features a clean user interface, interactive product cards, and a dynamic cart system that updates instantly as items are added. It is designed as a beginner-friendly yet practical project to demonstrate core front-end development skills.

🚀 Features
🛍️ Product listing with images and prices
➕ Add to cart functionality
🧮 Real-time cart updates
💻 Responsive and modern UI design
⚡ Fast and lightweight (no backend required)
🛠️ Tech Stack
HTML
CSS
JavaScript
🎯 Purpose

This project is created for learning and showcasing front-end development skills and can be extended into a full-stack e-commerce platform.
